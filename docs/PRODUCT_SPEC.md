# 日拱一卒 - 产品方案 V1.0

> 日期：2026-04-22  
> 状态：原型设计阶段

## 一、产品定位

**「日拱一卒」** — 目标驱动的日程管理 APP

核心理念：把大目标拆成每天可执行的小任务，每天反思、规划、复盘，形成成长飞轮。

**口号：日拱一卒，功不唐捐**

---

## 二、核心功能模块

### 2.1 日程管理（首页 Tab）

**视图模式：**
- **日视图**：全天时间块（6:00-24:00），任务以色块占据时间段
- **周视图**：7天横排，每天显示任务缩略
- **月视图**：日历视图，标记有任务的日期
- 支持纵向缩放（调整每小时高度）

**交互：**
- 左右滑动切换日期
- 点击空白时段 → 自动填充开始时间 → 快速添加任务
- 任务可标记完成/推迟/取消
- 底部显示空闲时段汇总

### 2.2 目标管理（第二 Tab）

**列表视图：**
- 顶部甘特图总览（多目标进度横向对比，红色竖线标记今天）
- 每个目标一行：名称 + 彩色进度条 + 百分比 + 剩余天数 + 截止日期

**多级拆分：**
- 目标 → 子目标 → 任务（无限层级树状结构）
- 点击展开/折叠查看子任务
- 每个任务右侧有 📅 按钮，一键添加到日程
- 添加子任务时可勾选"同时添加到日程"

**属性：**
- 截止日期、优先级（高/中/低）、颜色标签
- 状态：进行中 / 已完成
- 自动计算剩余天数

### 2.3 复盘中心（第三 Tab）

**数据看板：**
- 统计卡片：本周完成率、连续天数、本周任务数、进行中目标
- 完成率趋势折线图（近7天）
- 未完成原因饼图
- 活跃热力图（近4周）

**历史记录：**
- 每日复盘记录卡片列表
- 显示日期、完成率、心情、一句话总结

### 2.4 晨间反思（日程页触发）

**模板系统：**
- 内置模板：
  - 每日反思（最重要的3件事 + 昨天值得保持的 + 心情 + 一句话）
  - 每周回顾（本周成就 + 最大收获 + 改进点 + 下周目标）
- 自定义模板：用户自建，保存后可复用
- 模板包含可配置的问题列表，可自由添加/删除问题

**提醒：** 每天 8:00

### 2.5 晚间复盘（日程页触发）

**模板系统：**
- 内置模板：
  - 快速复盘（任务逐项标记 ✅⏭️❌ + 一句话总结）
  - 详细复盘（做得好的 + 需改进的 + 时间分配评估 + 明天调整）
  - KPT法（Keep/Problem/Try）
- 自定义模板：同反思
- 对比今日计划 vs 实际完成

**未完成原因分类：** 时间不够 / 被打断 / 动力不足 / 难度超预期 / 计划不合理

**提醒：** 每天 21:00

### 2.6 记录与检索

- 所有反思和复盘的历史列表
- 按日期分组展示
- 全文检索（搜索内容关键词高亮）
- 类型筛选（全部/反思/复盘）
- 点击查看详情弹窗

### 2.7 模板管理

- 查看所有模板（内置 + 自定义）
- 创建新模板（名称、图标、类型、问题列表）
- 编辑/删除自定义模板
- 内置模板不可删除

### 2.8 我的页面

- 用户信息
- 提醒时间设置
- 主题设置
- 数据导出
- 反思复盘记录入口
- 模板管理入口
- 关于

## 三、信息架构

```
底部Tab:  📅 日程 | 🎯 目标 | 📊 复盘 | 👤 我的

日程页:
├── 日期选择器（左右切换）
├── 视图切换（日/周/月）
├── 晨间反思入口（早上）
├── 时间块视图（任务色块）
├── 空闲时段汇总
└── 晚间复盘入口（晚上）

目标页:
├── 甘特图总览
├── 目标列表（进度条）
├── 目标展开（多级拆分树）
│   └── 每个任务有 📅 一键加到日程
└── 添加新目标

复盘页:
├── 统计卡片
├── 图表（折线/饼图/热力图）
└── 每日复盘记录

我的页:
├── 反思复盘记录（列表+搜索）
├── 模板管理
├── 设置（提醒/主题/导出）
└── 关于
```

## 四、数据模型

### Goal（目标）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |
| title | String | 目标名称 |
| description | String | 描述 |
| parentId | String? | 父目标ID（支持多级嵌套） |
| deadline | DateTime? | 截止日期 |
| priority | Enum | 高/中/低 |
| color | String | 颜色标识 |
| status | Enum | 进行中/已完成 |
| progressPct | int | 完成百分比 |
| createdAt | DateTime | 创建时间 |

### Task（日程任务）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |
| title | String | 任务名称 |
| date | DateTime | 日期 |
| startTime | double | 开始时间（小时，如 9.5 = 9:30） |
| duration | double | 时长（小时） |
| goalId | String? | 关联目标ID |
| status | Enum | pending/done/postponed/cancelled |
| color | String | 颜色标识 |
| notes | String? | 备注 |

### ReflectionTemplate（反思/复盘模板）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |
| name | String | 模板名称 |
| type | Enum | reflection/review |
| icon | String | 图标 |
| isBuiltIn | bool | 是否内置 |
| questions | List<String> | 问题列表 |

### JournalEntry（反思/复盘记录）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |
| type | Enum | reflection/review |
| templateId | String | 使用的模板ID |
| templateName | String | 模板名称 |
| date | DateTime | 日期 |
| time | String | 时间 |
| content | String | 内容 |
