# 日拱一卒 APP - 开发需求文档

## 概述
目标驱动 + 日程管理 + 复盘闭环 的移动端APP。

## 技术栈
- Flutter (跨平台 iOS/Android)
- 本地存储：Hive 或 SQLite
- 状态管理：Provider 或 Riverpod

## 核心功能

### 1. 日程管理（首页 Tab）
- **日视图**：全天时间块（6:00-24:00），任务以色块占据时间段，空闲时段一目了然
- **周视图**：7天横排，每天显示任务缩略
- **月视图**：日历视图，标记有任务的日期
- 点击空白时段添加任务（自动填充开始时间）
- 任务可标记完成/推迟/取消
- 左右滑动切换日期
- 纵向缩放（调整每小时高度）

### 2. 目标管理（第二 Tab）
- 顶部甘特图总览（多目标进度对比）
- 目标列表：每个目标一行（名称 + 进度条 + 百分比 + 剩余天数）
- **多级拆分**：目标 → 子目标 → 任务（无限层级树状结构）
- 点击展开查看子任务
- 每个任务有 📅 按钮，一键添加到日程
- 添加子任务时可选择"同时添加到日程"
- 目标状态：进行中 / 已完成
- 截止日期、优先级

### 3. 复盘中心（第三 Tab）
- 统计卡片：完成率、连续天数、本周任务数、进行中目标
- 完成率趋势折线图
- 未完成原因饼图
- 活跃热力图
- 每日复盘记录列表

### 4. 晨间反思（日程页触发）
- 模板化反思：
  - 内置模板：每日反思、每周回顾
  - 自定义模板：用户自建，保存后可复用
  - 模板包含可配置的问题列表
- 每天早上 8:00 提醒

### 5. 晚间复盘（日程页触发）
- 模板化复盘：
  - 内置模板：快速复盘、详细复盘、KPT法
  - 自定义模板：用户自建，保存后可复用
- 对比今日计划 vs 实际完成
- 逐项标记完成/推迟/放弃
- 未完成原因选择
- 每天 21:00 提醒

### 6. 记录与检索（我的页入口）
- 所有反思和复盘的历史列表
- 按日期分组
- 全文检索（搜索内容、模板名、日期）
- 类型筛选（全部/反思/复盘）
- 点击查看详情

### 7. 模板管理（我的页入口）
- 查看所有模板（内置+自定义）
- 创建新模板（名称、图标、类型、问题列表）
- 编辑/删除自定义模板
- 内置模板不可删除

### 8. 我的页面
- 用户信息
- 提醒时间设置
- 主题设置
- 数据导出
- 反思复盘记录入口
- 模板管理入口

## 数据模型

### Goal
- id, title, description, deadline, priority, color, status
- parentId (nullable, 支持多级嵌套)
- progressPct

### Task (日程任务)
- id, title, date, startTime, duration
- goalId (nullable, 关联目标)
- status (pending/done/postponed/cancelled)
- color, notes

### ReflectionTemplate
- id, name, type (reflection/review), icon
- isBuiltIn, questions[]

### JournalEntry (反思/复盘记录)
- id, type (reflection/review)
- templateId, templateName
- date, time
- content
